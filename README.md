# FBrowser: A Custom Web Browser Built with .NET Windows Forms and WebView2

FBrowser is a lightweight, custom-built web browser using .NET Windows Forms and Microsoft WebView2. The application supports essential browser functionalities such as navigation, tab management, bookmarks, and history tracking, while offering a user-friendly interface.

## Features
- **Tabbed Browsing**: Open multiple tabs to browse different websites simultaneously.
- **Address Bar**: Quickly navigate to any website using the address bar.
- **Navigation Controls**: Includes back, forward, refresh, and home buttons for easy navigation.
- **Bookmarks**: Save your favorite websites and access them anytime via the bookmarks manager.
- **History Management**: Automatically saves browsing history and displays it in a separate window.
- **Database Integration**: Stores bookmarks and history using SQL Server for persistent data management.

## Technologies Used
- **.NET Windows Forms**: For building the application interface.
- **Microsoft WebView2**: For rendering web content.
- **SQL Server**: For storing and managing bookmarks and history.
- **C#**: The primary programming language for this project.

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/FBrowser.git
   ```
2. Open the solution file (`FBrowser.sln`) in Visual Studio.
3. Restore NuGet packages and build the project.
4. Ensure you have Microsoft WebView2 Runtime installed on your system. You can download it [here](https://developer.microsoft.com/en-us/microsoft-edge/webview2/).
5. Configure the SQL Server connection string in the `sqlbaglantisi` class.
6. Run the application.
